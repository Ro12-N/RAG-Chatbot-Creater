# RAG Chatbot – YouTube & Instagram Video Analyzer

A full-stack RAG chatbot that analyzes and compares YouTube videos and Instagram Reels using LangChain, FastAPI, ChromaDB, and OpenAI.

## Tech Stack
- Backend: FastAPI, LangChain, ChromaDB, OpenAI
- Frontend: React, Vite
- Transcription: Whisper, yt-dlp, youtube-transcript-api
